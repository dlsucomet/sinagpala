## Pre-requisites

- Node.js

## Installation
Open the Sinagpala site folder and run the following commands:

1.  **Running locally in development mode**

    ```shell
    npm i
    npm run dev
    ```
    The server will run on localhost:3000

2.  **Building and deploying in production**
    ```shell
    npm i
    npm run build
    npm run start
    ```
